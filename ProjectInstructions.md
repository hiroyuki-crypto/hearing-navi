# ヒアリングナビ 開発指示書

## プロジェクト概要

不動産営業のヒアリングを自動化するAI会話システム。
パートスタッフでも営業ができるよう、AIが会話をリードする。

---

## 会社・商材情報

**会社名：** 株式会社市兵衛 / 株式会社チームアンド

**4つの商材：**
| 商材名 | ターゲット |
|---|---|
| キャンディハウス | 所得が高くない・新築希望 |
| 平屋マーケット | 高齢者・古い家・子供が独立済み |
| イオンハウジング藤井寺（中古＋リフォーム） | 新築が買えない・中古でもOK |
| 戸建賃貸 | 土地持ち・相続・投資に興味あり |

---

## システムの使い方（運用フロー）

1. パートスタッフがiPad/PCをお客さんの前に置く
2. 画面にAIが考えた「質問文」が表示される
3. パートスタッフがその質問をお客さんに**読み上げる**
4. お客さんが答える
5. スタッフが**マイクボタンを押して**お客さんの返答を録音
6. 音声が自動で文字起こしされる
7. AIが内容を理解して**次の質問を自動生成**
8. これを繰り返し、AIが商材を自動判定
9. 最後にクロージングトークが画面に表示される
10. スタッフはそれを読み上げるだけ

**ポイント：スタッフは読み上げるだけ。判断は一切不要。**

## 技術仕様

### 使用技術
- **HTML / CSS / JavaScript**（単一ファイル）
- **Web Speech API**（ブラウザ標準の音声認識・文字起こし）
- **Anthropic Claude API**（claude-sonnet-4-20250514）
- GitHubPages でホスティング

### APIの使い方

```javascript
const response = await fetch("https://api.anthropic.com/v1/messages", {
  method: "POST",
  headers: {
    "Content-Type": "application/json",
    "x-api-key": API_KEY,
    "anthropic-version": "2023-06-01"
  },
  body: JSON.stringify({
    model: "claude-sonnet-4-20250514",
    max_tokens: 1000,
    system: SYSTEM_PROMPT,
    messages: conversationHistory
  })
});
```

### 会話履歴の管理
- 会話履歴を配列で保持し、毎回APIに全履歴を渡す
- `{ role: "user", content: "..." }` と `{ role: "assistant", content: "..." }` を交互に追加

## AIへのシステムプロンプト

```
あなたは不動産会社「株式会社市兵衛」のAIヒアリングアシスタントです。

【あなたの役割】
パートスタッフがお客様にあなたの質問を読み上げます。
お客様の回答をもとに、以下の4商材のどれが最適かを判断してください。

【4つの商材】
1. キャンディハウス：ローコスト新築住宅（所得が高くない・新築希望）
2. 平屋マーケット：平屋の建て替え（高齢者・古い家・子供が独立済み）
3. イオンハウジング藤井寺：中古＋リフォーム（新築が買えない・中古でもOK）
4. 戸建賃貸：土地活用・相続対策（土地持ち・相続・投資に興味あり）

【ヒアリングの進め方】
- 1回の返答に対して1つの質問を返す
- 自然な会話になるよう、お客様の言葉に共感してから次の質問をする
- 失礼にならない丁寧な言葉遣いで
- 質問は5〜8回程度で商材を判定できるよう効率よく進める
- 必ず確認すること：家族構成・現在の住まい・予算感・土地の有無・きっかけ・時期

【出力フォーマット】
通常のヒアリング中は以下のJSON形式で返答すること：
{
  "type": "question",
  "question": "スタッフが読み上げる質問文",
  "memo": "スタッフ向けの補足メモ（表示はするが読み上げない）"
}

商材が判定できたら以下のJSON形式で返答すること：
{
  "type": "result",
  "product": "商材名",
  "reason": "判定理由（スタッフ向け）",
  "closing": "クロージングトーク全文（スタッフが読み上げる）",
  "scores": {
    "キャンディハウス": 数値,
    "平屋マーケット": 数値,
    "イオンハウジング藤井寺": 数値,
    "戸建賃貸": 数値
  }
}

必ずJSONのみを返すこと。前置きや説明は不要。
```

## 重要な注意事項

- **APIキーは画面上部の入力欄から入力できるようにする**（ハードコードしない）
- Web Speech APIはChromeのみ対応。Safariは非対応なので注意書きを表示
- iPadでも使えるようにタッチ操作に最適化（ボタンは大きく）
- マイクボタンは押している間だけ録音するプッシュトゥトーク方式
- 通信エラー時は「もう一度試してください」と表示
- 会話履歴はページを閉じるまで保持

## デザイン要件

- シンプルで見やすい
- 文字は大きく（スタッフが読み上げやすいように）
- 質問文は特に大きく表示（最低24px以上）
- カラー：オレンジ系（#D9622B）をメインカラーに
- スマホ・タブレット対応（レスポンシブ）
